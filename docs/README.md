## 🚀 クイックスタート

### WebFOCUSでPython関数を使うための3ステップ

1. **Pythonアダプタの設定** - WebFOCUS管理コンソールでPythonアダプタを追加
2. **Python関数の作成** - CSV入出力を扱う関数を作成
3. **シノニムの作成** - WebFOCUSから関数を呼び出すためのメタデータを生成

詳細は各ドキュメントを参照してください。

## 📋 前提条件

- WebFOCUS Reporting Server (バージョン8.2以降推奨)
- Python 3.6.x (WebFOCUSサーバと同じビットサイズ)
- 必須パッケージ: numpy, scipy, scikit-learn, pandas
